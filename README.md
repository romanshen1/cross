## Интернет-магазин Кроссовок
Описание
## Используемые технологии
- PHP 7.2
- Apache2
- MySQL
- OpenServer

# Информация по установке
- cd domains
- git clone
- composer self-update
- composer install
- создать файл .env из .env example
- создать базу с именем cross и перейти в нее
- импорт файла cross.sql
- перезапуск сервера
